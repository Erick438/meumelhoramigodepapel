DESINFORMAÇÃO, de Ion Mihai Pacepa

Ficha técnica

  • Título completo: Desinformação: Ex-chefe de espionagem revela estratégias secretas para solapar a liberdade, atacar a religião e promover o terrorismo
  • Autor: Ion Mihai Pacepa e Ronald J. Rychlak
  • Editora: Vide Editorial
  • Ano de publicação: 2015
  • Número de páginas: 556 páginas
  • ISBN: 978-8567394732

A obra, escrita entre duas mãos, versa sobre o tema de como os sistemas de inteligência têm a capacidade de forjar a mentalidade de uma nação inteira. É bastante esclarecedor, particularmente quando se aborda a definição dos termos "glasnost" e "enquadramento", duas táticas bastante comuns na para a KGB. O livro possui algo de autobiográfico: Pacepa foi um agente da Securitate, por assim dizer, uma filial da KGB na Romênia. Demonstrando ter grande conhecimento histórico dessas instituições, ele intercala relato autobiográfico com referências de outros estudiosos. O leitor poderá enxergar no texto o "mea culpa" do autor principal, mas ao mesmo tempo conferir como a desinformação, uma informação às avessas, funciona como um método experimental em que as cobaias são os indivíduos da sociedade.

Recomendo vivamente esse livro a todos os que se interessam por temas como história, política, literatura e sistemas de espionagem. Uma leitura inesquecível e bastante enriquecedora.

Getting Started

Install Dependencies

In order to have Wok running as expected in your system, please make sure to have all the dependencies installed before building Wok or starting up the wokd service.

  • RHEL/Fedora systems
  • Debian/Ubuntu systems
  • openSUSE systems

Build and Install

$ ./autogen.sh --system
$ make

# Optional if running from the source tree
$ sudo make install

Starting up Wok

$ sudo python src/wokd

To access Wok, please, connect your browser to https://localhost:8001.
